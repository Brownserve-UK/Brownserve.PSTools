# New-TempDirectory

## SYNOPSIS
Creates a new temporary directory with a random name in the systems temporary directory

## SYNTAX

```
New-TempDirectory [<CommonParameters>]
```

## DESCRIPTION
Creates a new temporary directory with a random name in the systems temporary directory

## EXAMPLES

### Example 1
```powershell
PS C:\> New-TempDirectory
```

Would create a new temp directory in C:\Windows\temp
